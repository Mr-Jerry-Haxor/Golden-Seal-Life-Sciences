Privacy and Data Collection

This project uses consent-based analytics. Visitors can opt in through the privacy banner. Anonymous event metadata is stored in Firebase under analytics_events.

Contact details are only captured via explicit form submission with user consent and are stored in Firebase under leads.

All tracking and communication flows are intentionally implemented for transparent and ethical compliance.
